# Media Types

Engine knowledge. Generic to any user. The Media Scripter and the Media Creator load this. Each type has a use, design guidance, and the prompt shape it needs.

## Single image

Use, hook visual, a quote, a single stat, a meme. Design, high contrast, bold type, few words, a human element if it fits. Prompt shape, subject, style, text overlay, color, mood.

## Document carousel, the PDF

Use, step by step guides, stories, data breakdowns, listicles. The highest reach format. Design, slide one is the bold hook, middle slides one idea each, last slide a soft follow prompt, not an outbound link. Keep one font set, one palette, one layout grid across slides. If a character or avatar appears, keep it consistent slide to slide. Prompt shape, per slide visual, text overlay, placement.

## Video

Use, talking head, screen recording, animated explainer, text overlay reel. Strong format reach, and captions are needed because most play silent. Workflow, scenes with timings, on screen text, motion, and a caption file. Prompt shape, scene by scene visual plus audio direction plus caption text per timestamp.

## GIF or animated image

Use, humor, reaction, a short demo. Prompt shape, start frame, end frame, loop, duration, emotion.

## Poll

No visual asset needed, the copy carries the question and the options. Strong for early comments.

## Link post

Note, outbound links cut reach hard, prefer to avoid. If a link is needed, a custom preview image matters, sized to the platform standard, with the title and brand on it.

## How to use this

The strategy brief sets the media type, match it. The media amplifies the copy, it does not echo it. Keep the owner's brand system consistent, the Art Director owns that. Never put an outbound link inside the asset.
