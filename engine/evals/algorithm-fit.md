# Eval, Algorithm Fit

The Distribution Strategist runs this before handing off. Every item must pass.

- [ ] A post type and media type are recommended for the chosen topic
- [ ] Each call is justified with a current mechanic from the algorithm file
- [ ] The owner's own results in posts.csv were checked, and override the general rule where they disagree
- [ ] The format favours dwell time and early comments for this topic
- [ ] Reach risks are flagged, for example an outbound link penalty, with a way around them
- [ ] A best post time is noted if the data supports one

Pass only if every box is checked.
