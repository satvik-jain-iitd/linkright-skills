# Eval, Asset Quality

The Asset Builder runs this on the rendered asset. Every item must pass.

- [ ] The asset matches the media type the script set
- [ ] The locked design system was applied, palette, type, grid, watermark
- [ ] The asset amplifies the copy, it does not just restate it
- [ ] A carousel cover stops the scroll, middle slides carry one idea each, last slide is a soft follow prompt
- [ ] No outbound link inside the asset
- [ ] Rendered in the sandbox, the external fallback used only if the render fell short, and noted
- [ ] Files saved to the run folder and listed in the handoff

Pass only if every box is checked.
