# Eval, Fingerprint Completeness

The Stylometrist runs this on the metrics before handing off. Every item must pass.

## Measurement gates

- [ ] Function word habits recorded, the repeated grammatical words with examples
- [ ] Sentence length distribution recorded as a spread, not just an average
- [ ] Burstiness recorded, the range and whether the owner swings or stays even
- [ ] Punctuation profile recorded, confirmed against config and checked against real samples
- [ ] Signature phrases and n grams pulled verbatim with a rough frequency
- [ ] Openers and closers noted

## Evidence gates

- [ ] Every claimed marker carries a real verbatim example and a rough count
- [ ] Stable markers separated from one off quirks
- [ ] Thin signals labelled thin, nothing invented to fill a gap
- [ ] Source noted per marker, operational chat distinguished from reflective writing

## Result

Pass only if every box is checked. On a miss, return to the samples and complete the measure.
