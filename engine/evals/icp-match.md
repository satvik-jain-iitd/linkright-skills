# Eval, ICP Match

The Audience Mapper runs this before handing off. Every item must pass.

- [ ] Each candidate topic is tagged with one named primary reader, not a demographic blur
- [ ] The exact pain or desire the topic touches for that reader is named
- [ ] At least one chosen candidate serves a primary or secondary reader the owner wants
- [ ] Reader segments came from the instance ICP file, not invented
- [ ] Any topic that serves no target reader is flagged as off target

Pass only if every box is checked.
