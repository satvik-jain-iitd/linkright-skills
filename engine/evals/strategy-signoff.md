# Eval, Strategy Signoff

The Strategy Reviewer runs this gate after W2. A pass lets drafting start. Any fail returns the brief.

- [ ] The brief serves the owner's deep goal, read from config
- [ ] The primary reader and the pain are specific and real
- [ ] The topic ladders to the positioning through line
- [ ] The angle is different enough from the last several posts, no repeat
- [ ] The post type and media type fit the topic and the current mechanics
- [ ] No obvious risk in the angle, teardowns of real companies are flagged for the risk gate
- [ ] Every brief field is filled, nothing thin

Result, PASS only if every box is checked, else REWORK with the exact gap named.
