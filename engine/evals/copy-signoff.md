# Eval, Copy Signoff

The Copy Reviewer runs this gate after W3. A pass lets media production start. Any fail returns the draft.

## The three lenses

- [ ] Scroll stopper, the open breaks the pattern inside the mobile fold
- [ ] Trust builder, the post makes the reader believe the author, evidence and a real voice
- [ ] Action driver, it gives a real reason to comment, save, or follow

## Floors

- [ ] Voice authenticity and human texture both clear the rubric threshold
- [ ] Hard gates already passed at drafting, confirmed not reintroduced
- [ ] Reads like the owner, not generic

Result, PASS only if every box is checked, else REWORK with the failing lens named.
