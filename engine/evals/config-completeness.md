# Eval, Config Completeness

The Onboarding Concierge runs this before the config is locked. It also gates the W0 onboarding output and the setup chain before voice building starts. Every item must pass.

- [ ] User block filled, name, background, goal
- [ ] Source files listed, the project brief and ground truth
- [ ] Language set, input language and output language
- [ ] Writing protocol set, allowed and banned punctuation, banned words, hook rules, paragraph shape
- [ ] Positioning set, the through line and framing rules
- [ ] Expertise set, the domain the Domain SME adapts to
- [ ] Cadence set
- [ ] Audience at least roughed in, primary reader named
- [ ] Metrics policy and consents recorded
- [ ] No required field left blank, gaps were asked, not guessed

Pass only if every box is checked.
