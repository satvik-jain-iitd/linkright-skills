# Eval, Media Script Clarity

The Media Scripter runs this before handing off. Every item must pass.

- [ ] The media type matches the one set in the strategy brief
- [ ] The script gives the Media Creator enough to build without guessing intent
- [ ] A carousel is laid out slide by slide, cover hook, one idea per slide, soft follow prompt on the last
- [ ] A video has timed scenes, on screen text, and caption notes
- [ ] The media amplifies the copy, it does not just restate it
- [ ] No outbound link is placed inside the asset

Pass only if every box is checked.
