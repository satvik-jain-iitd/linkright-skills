# Eval, Positioning Ladder

The Positioning Guard runs this before handing off. Every item must pass.

- [ ] Each candidate is checked against the owner's through line, read from config and the positioning file
- [ ] The recent pillar mix was reviewed from posts.csv
- [ ] The recommended pillar fills a thin one or keeps the mix even
- [ ] Any topic that fights the positioning is flagged, even if it would perform
- [ ] No trend chase that pulls the brand off its arc was approved

Pass only if every box is checked.
