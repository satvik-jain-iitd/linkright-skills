# Eval, Accessibility

The Accessibility agent runs this before handoff. Every item must pass.

- [ ] Every image or slide has clear alt text, describing the content and the point
- [ ] Any video has timestamped captions or an srt
- [ ] Alt text is honest and useful, not keyword stuffed
- [ ] Text on the asset has enough contrast to read on a phone
- [ ] Any low contrast text was flagged for the Art Director

Pass only if every box is checked.
