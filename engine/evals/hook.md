# Eval, Hook

The Hook Writer runs this. Every item must pass.

- [ ] The hook fits inside the mobile cutoff from config
- [ ] It opens on a hard fact or a contrast, the owner's opening move
- [ ] A multi sentence hook is split onto its own lines, each beat visible
- [ ] The last line ends on the configured signature, exactly
- [ ] The body delivers on what the hook promises, no bait and switch
- [ ] One alternative hook is offered when the brief is open

Pass only if every box is checked.
