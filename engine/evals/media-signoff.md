# Eval, Media Signoff

The Media Reviewer runs this gate after W4. A pass lets the post move to publish and engage. Any fail returns the asset.

## The three lenses

- [ ] Thumb stopper, you would stop scrolling for this in the feed
- [ ] Brand ambassador, it looks like the owner's other work, one consistent system
- [ ] Story visualizer, the visual carries the point without the caption

## Floors

- [ ] The asset matches the copy and the chosen media type
- [ ] Alt text and captions are present
- [ ] No outbound link inside the asset

Result, PASS only if every box is checked, else REWORK with the failing lens named.
