# Eval, Risk

The Ethics and Risk Reviewer runs this alongside the copy and media gates. Any fail blocks the post until fixed.

- [ ] Any teardown of a real company is fair comment on sourced facts, not an unbacked accusation
- [ ] Sharp claims about a named party are sourced or clearly framed as opinion
- [ ] No invented quote attributed to a real, named person
- [ ] Every number is real and sourced, or a locked synthesized figure
- [ ] No number contradicts the ledger
- [ ] The owner's own career numbers are accurate, not inflated

Result, PASS only if every box is checked, else block with the exact risky line and the soften or source fix.
