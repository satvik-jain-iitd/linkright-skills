# Eval, Credibility and Metrics

The Domain SME runs this before handing substance to the Storyteller. Every item must pass.

- [ ] The substance is specific and expert, not a generic take
- [ ] Real numbers from the owner's source files are used wherever history supports them
- [ ] Any synthesized number is plausible and clearly labelled for the ledger
- [ ] Every number matches the ledger, no contradiction with past posts
- [ ] Any new synthesized number is appended to the ledger with its context
- [ ] No invented job or company, only texture around real projects

Pass only if every box is checked.
