# Eval, Differentiation and Brief Completeness

The Topic Selector runs this before handing off. Every item must pass.

## Differentiation

- [ ] Candidates were ranked on a clear composite, goal fit, audience match, and difference from recent posts
- [ ] The last several posts in posts.csv were checked, the pick is not a repeat of a recent topic or angle
- [ ] The pick was not flagged off brand by the Positioning Guard

## Brief completeness

- [ ] Every field of the strategy brief is filled, no blanks
- [ ] The angle is specific, not the broad topic
- [ ] Notes for the drafter and notes for the media agent are both present
- [ ] Any metric to be used is named and matches the ledger

Pass only if every box is checked.
