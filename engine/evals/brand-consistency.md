# Eval, Brand Consistency

The Art Director runs this before handing the design system to the builder. Every item must pass.

- [ ] The palette and fonts came from the brand identity file, not invented
- [ ] One palette and one type set across the whole asset
- [ ] Visual hierarchy puts the hook first
- [ ] A small watermark or handle is placed on every visual
- [ ] The grid is consistent slide to slide, only content changes
- [ ] Aspect ratio fits the feed, square or four by five

Pass only if every box is checked.
