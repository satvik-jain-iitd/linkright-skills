# Eval, First Comment

The First Comment agent runs this. Every item must pass.

- [ ] The comment is written in the owner's voice
- [ ] It asks one real, specific question tied to the post
- [ ] It extends the post, it does not restate it
- [ ] It follows the writing rules, same gates as the post
- [ ] It reads inviting, not like desperate engagement bait

Pass only if every box is checked.
