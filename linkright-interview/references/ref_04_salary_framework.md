# ref_04 — Salary Negotiation Framework

## Core Principles

1. **Never give a number first** if you can avoid it
2. **Anchor high** — your first number should be above target, not at target
3. **Silence is leverage** — make your ask, then stop talking
4. **Get to written offer before negotiating** — verbal offers don't bind anyone
5. **Negotiate total comp, not just base** — equity and bonus are levers too

---

## Script Library

### When Asked "What's Your Salary Expectation?"

**Before offer (early interview stage):**
```
"I'm keeping an open mind at this stage — I want to make sure we're a good fit first.
Could you share the band for this role so I can make sure we're aligned?"
```

**If they push:**
```
"I'd rather not anchor early. My expectation is that [company] pays competitively
for this level — if we get to the offer stage I'm confident we can find a number
that works for both of us."
```

**If they require a number:**
```
"Based on my research for [role] in [market], I'd expect total comp
in the range of [target_ctc_target to target_ctc_target × 1.15].
I'm flexible depending on the full package."
```

---

### When Receiving a Verbal Offer

Never accept verbally. Always:
```
"Thank you — I'm really excited about this. I'd love to see it in writing
so I can review the full picture. Can you send over the offer letter?"
```

---

### Opening Negotiation (after written offer)

**If offer is within range:**
```
"I'm really excited about this role and [company]. The offer is close —
to make it an easy yes, is there room to get to [target_ctc_target] on base?"
```

**If offer is below floor:**
```
"I appreciate the offer. To be transparent, this is below where I need to be
to make a move. I was expecting [target_ctc_target]. Is there flexibility
to get there on base or equity?"
```

**Equity-focused ask (when base is fixed):**
```
"I understand base budgets can be constrained. If there's more flexibility
on equity, that would make a real difference — could we look at
[specific equity ask, e.g., 20% more options or accelerated vesting]?"
```

---

### When They Push Back

**"That's at the top of our band":**
```
"I understand. Can we look at the other components —
is there flexibility on the signing bonus or an earlier first review?"
```

**"We don't negotiate":**
```
"I appreciate you being direct. Would you be open to a performance review
at 6 months with a clear target for a raise? That would help bridge the gap."
```

**"We need an answer by [date]":**
```
"I respect the timeline. I'm very close to a decision — could I have
until [date + 2 days] to review the full package with my family?"
```

---

### Asking for an Extension

```
"I'm very interested and taking this seriously. I have one other conversation
at a late stage — could I have until [date] to give you a final answer?
I want to make this decision thoughtfully."
```

Never reveal the other company unless it's a real leverage point.

---

## Escalation Path

```
Step 1: Ask for target on base
  ↓ rejected
Step 2: Ask for equity top-up (additional RSUs or options)
  ↓ rejected
Step 3: Ask for signing bonus (one-time, budget-friendly for employer)
  ↓ rejected
Step 4: Ask for 6-month review with stated performance targets + raise trigger
  ↓ rejected
Step 5: Evaluate — is this below floor? If yes, decline with clear reasoning.
        If within range and you want it, accept and document the decision.
```

## Anchoring Rules

| Your target | Your opening anchor |
|---|---|
| X | X × 1.10 to 1.15 |
| Floor | Never use floor as anchor — use target |

Example: target = ₹60L, floor = ₹45L
→ Open at ₹66-70L. If they counter ₹55L, you end up negotiating toward ₹60L.
→ If you open at ₹60L and they push back, you end up at ₹55L or lower.

## Comp Anatomy Cheatsheet

| Component | Notes |
|---|---|
| Base salary | Fixed, most predictable — primary negotiation lever |
| Annual bonus | Target % of base — verify if it's guaranteed or discretionary |
| RSUs / stock options | 4-year vest standard — verify cliff, acceleration clause |
| Signing bonus | One-time — may have clawback if you leave <1yr |
| Benefits | Health, 401k match, equity refresh cycle |

Total comp = base + (equity / 4yr annualized) + target bonus

Do NOT include signing bonus in total comp comparison — it's not recurring.

## When to Walk Away

Walk away if all of these are true:
1. Offer is below target_ctc_min after all negotiation levers exhausted
2. No performance review or raise trigger available within 6-12 months
3. Role trajectory is lateral or backward (see ref_04_offer_evaluation.md)

Document in decision journal before declining. Note the key assumption.
